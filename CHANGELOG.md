## CR-001: Governance Manual Reform — Terminology & CoP/CoE Alignment

**Date:** 2026-03-04
**Requested by:** Project Manager (Vitor Rodovalho)
**Status:** 🟡 PROPOSED
**Priority:** HIGH
**Affects:** Governance Manual R2 (DocuSign signed, Sep 2025)

### Background

Strategic analysis (consolidated report, March 2026) identified that the Núcleo operates as a hybrid Center of Excellence (CoE), not a classical Community of Practice (CoP). To position for international expansion and eventual AI Ambassadors program alignment with PMI Global, the Manual requires targeted reforms.

### Requested Changes

| # | Change | Current State | Proposed State | Rationale |
|---|--------|--------------|----------------|-----------|
| 1.1 | Add bilingual name | Only PT-BR | Add EN-US official name: "The AI & PM Study and Research Hub" | International recognition |
| 1.2 | Adopt CoP/CoE terminology | "Núcleo de Estudos e Pesquisa" | Self-describe as "Center of Excellence" in governance docs; create Axis B as "Community of Practice" | PMI Global vocabulary alignment |
| 1.3 | Rename "Tribos" in EN materials | "Tribos" / "Tribes" | "Research Streams" | International academic convention |
| 1.4 | Add trilingual abstract | PT-BR only | EN-US + ES-LATAM summary sections | Global accessibility |
| 1.5 | Formalize rotation mechanism | Not documented | Partial leadership rotation at Levels 3-4 between cycles | CoP best practice (Wenger, PMI DA) |
| 1.6 | Create observer tier | Not available | Open "Observer" participation level for PMI members (embryo of Axis B / CoP) | Scalability without diluting quality |
| 1.7 | Reference PMI-CPMAI | Not mentioned | Connect research stream themes to PMI-CPMAI competencies | Certification ecosystem alignment |
| 1.8 | Add compliance section | Implicit | Explicit reference to Policy Manual for PMI Chapters (July 2024) and brand usage guidelines | Legal/brand protection |
| 1.9 | RAI & data protection scope | LGPD only | Add GDPR and AI Act considerations for international expansion | Global compliance readiness |
| 1.10 | LLM transparency clause | Implicit in ethics | Explicit disclosure requirements for AI-generated content in publications | Scientific integrity |

### Impact Assessment

- **Scope:** Expands project scope to include international dimension and Bimodal Model
- **Schedule:** Can be phased — items 1.1-1.4 in Cycle 3, items 1.5-1.10 in Cycle 4
- **Cost:** Zero incremental cost (documentation changes only)
- **Quality:** Increases alignment with PMI Global standards
- **Stakeholders:** Requires approval from all 5 chapter leaderships (Level 1)

### Approval Path

1. Present to Cycle 3 kickoff meeting (March 2026) as strategic roadmap
2. Collect feedback from tribe leaders (Level 3) during Cycle 3
3. Submit formal R3 draft for Level 1 approval at end of Cycle 3
4. DocuSign new version with all 5 chapter signatures

## CR-002: Bimodal Operating Model — Axis A (CoE) + Axis B (CoP)

**Date:** 2026-03-04
**Requested by:** Project Manager (Vitor Rodovalho)
**Status:** 🟡 PROPOSED
**Priority:** HIGH
**Affects:** Organizational structure, Level 5 (Ambassadors)

### Background

Both independent strategic analyses converge on the same architectural solution: the "Scalability Paradox" — the rigorous structure that built credibility is also the bottleneck for global expansion. Resolution: Bimodal Model.

### Requested Changes

| # | Change | Description |
|---|--------|------------|
| 2.1 | Define Axis A (CoE) | Levels 1-4 continue as closed Center of Excellence with current rigor |
| 2.2 | Define Axis B (CoP) | Transform Level 5 (Ambassadors) into open Community of Practice accessible to any PMI member |
| 2.3 | Define interaction protocol | CoE produces frameworks → CoP operationalizes → feedback loop |
| 2.4 | Create "AI Ambassador Code of Conduct" | Produced by CoE, applied by CoP members |
| 2.5 | Map CoP to PMI DA standards | Align Axis B with Disciplined Agile CoP characteristics |

### Impact Assessment

- **Scope:** Major scope expansion — creates new organizational axis
- **Schedule:** Phase 3 of roadmap (H1 2027)
- **Cost:** May require hosting infrastructure for CoP platform
- **Stakeholders:** Requires PMI Global engagement strategy
