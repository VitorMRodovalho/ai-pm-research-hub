Lane de estratégia para os acordos de cooperação do Núcleo IA (missão do mentor Mário Trentim, jun/2026). Mantém os **dossiês estratégicos** colados à estratégia do Núcleo (verticais, flywheel, CPMAI) de que derivam.

## Separação de responsabilidades (não duplicar)
- **Estratégia (aqui, em docs):** o canvas reutilizável + um dossiê de 1 página por parceiro.
- **Negócios (na plataforma):** interações, follow-ups, estágio do deal vivem no CRM de parceiros (`manage_partner`, `log_partner_interaction`, `get_partner_pipeline`, `get_partner_followups`). É lá que a continuidade entre sessões/pessoas sobrevive.

## Arquivos
- `partnership_value_exchange_canvas.md` — a tese, a espinha de troca (give/get), os arquétipos, o tiering/sequência e a disciplina. **Ler primeiro.**
- `partner_dossier_TEMPLATE.md` — copiar para `partner_<slug>.md` por parceiro. Inclui o **bloco fixo de prova de autoridade ANSI** (standards de projeto + o de IA de jun/2026).
- `partner_<slug>.md` — um por alvo (a criar). Sequência sugerida no canvas §D: CEIA-UFG → AI Brasil → Inteli → Insper → ALUN/StartSE → Delta → Cubo/Link.
